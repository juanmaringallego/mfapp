# React Native Template

Este directorio contiene un ejemplo sencillo de código React Native para mostrar datos obtenidos desde un backend o base de datos.

## Archivos

- `App.js`: configura la aplicación y renderiza el componente `DataDisplay`.
- `components/DataDisplay.js`: realiza una petición HTTP a un endpoint y muestra el resultado en un `FlatList`.

## Uso

1. Copia el contenido de `template` dentro de tu proyecto React Native.
2. Actualiza la URL de la petición en `components/DataDisplay.js` para que apunte a tu backend.
3. Ejecuta la aplicación con Expo o React Native CLI como de costumbre.

Este código está pensado como punto de partida para integrar datos de una base de datos en la interfaz de usuario.
